Welcome to the Truck Overload Detector

Ensuring road safety through real-time truck weight monitoring.

How It Works

The Truck Overload Detector continuously monitors truck weights using IoT sensors and Django-powered dashboards. If a truck exceeds the legal weight limit (48 tons), the system issues alerts and logs violations in the admin panel.

Key Features

Real-Time Monitoring

Live updates on truck weights to detect overloads instantly.

Automated Alerts

Instant alerts when a truck exceeds the 48-ton limit.

Admin Dashboard

Manage truck data and monitor overload history in one place.
